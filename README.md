#  멀티플레이 페인트 총알 FPS (가제)

투명한 원숭이들이 흔적을 쫓아 물리 기반 페인트탄을 쏘고, 피격 시 스킨이 조금씩노출되는 '개인전 멀티플레이 FPS' 프로젝트입니다.

##  개발 환경
- **Engine:** Unity 6.3.11f1 (URP)
- **Network:** Photon PUN 2 (예정)

## 구현된 기능 (프로토타입1)
- `CharacterController` 기반 1인칭 이동 및 시점 회전 제어 완료
- `Rigidbody`를 이용한 포물선 탄도학 사격 프로토타입 구현 (`AddForce`)
- 1인칭 카메라 Culling Mask 설정 (본인 캐릭터 Culling Mask에서 Player레이어 숨김)
- Skin Width 최적화를 통한 점프 및 바닥 판정 버그 수정

## 앞으로의 계획
- [ ] 오브젝트 풀링(총알, 흔적, 데칼) 최적화 모델 구조로 변경
- [ ] 데미지에 비례해 텍스처를 노출시키는 다중 스킨 셰이더 적용
- [ ] Photon 연동 및 실시간 멀티플레이 핑 테스트 (가짜 총알 RPC 동기화)
- [ ] 존버 방지용 '5초 정지 시 페인트 누수 패널티' 시스템 구체화
